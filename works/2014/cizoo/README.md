# 囍樹設計官網

![2014_cizoo](./2014_cizoo.png)

囍樹設計成立於 2007 年，擅長打造品牌形象、企業識別，包含包裝設計、網頁設計、空間形象整合到指標系統，將概念融合到品牌細節中。成就的客戶包含法雅客、可可法朋、綠藤生機、饗食天堂等。

## 角色

擔任全端工程師，與平面設計師合作，參與使用者體驗規劃、互動細節的設計，以及前後台的前後端程式開發。

## 技術棧

* 後端：Node.js、MongoDB
* 前端：jQuery
